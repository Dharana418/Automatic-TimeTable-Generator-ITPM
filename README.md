The Automatic TimeTable Generator System is a web-based application developed to simplify and automate the process of creating university timetables. Instead of manually arranging schedules, the system uses advanced algorithms to generate efficient and conflict-free timetables.

This system is designed for academic environments such as universities and institutes like SLIIT. An admin user manages the entire system, including creating and controlling all other users such as lecturers, instructors, and coordinators. This centralized approach ensures better control, consistency, and easier management.

The application is built using modern technologies like React for the frontend, Node.js and Express for the backend, and PostgreSQL for the database. It provides an easy-to-use interface, real-time dashboard, and intelligent scheduling features to improve academic planning and reduce manual effort.

Overall, the system helps save time, minimize scheduling conflicts, and improve the efficiency of timetable management.
